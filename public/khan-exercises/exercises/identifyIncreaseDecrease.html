A function f is increasing on an interval if larger abscissas have larger values on that interval. In other words, given any two abscissas x_1 and x_2 in the interval with x_1 \leq x_2 we have f(x_1) \leq f(x_2)

A function f is decreasing on an interval if larger abscissas have smaller values on that interval. In other words, given any two abscissas x_1 and x_2 in the interval with x_1 \leq x_2 we have f(x_1) \geq f(x_2)
