If f(X) = Y, what point must be on the graph of f^{-1}?

( Y ,X)

By definition, f^{-1}(f(x)) = x for all x in the domain of f

So f^{-1}(f(X)) = X

Thus f^{-1}(Y) = X

So (Y, X) is on the graph of f^{-1}.

Notice that the points on a function and its inverse function have reversed abscissas and ordinates.
