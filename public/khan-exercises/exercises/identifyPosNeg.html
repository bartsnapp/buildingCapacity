A function f is positive on an interval if the function only returns values which are greater than or equal to zero on that interval. Symbolically f(x) \geq 0 for all x in the interval.

A function f is negative on an interval if the function only returns values which are less than or equal to zero on that interval. Symbolically f(x) \leq 0 for all x in the interval.

Graphically a function is positive if the graph lies above the horizontal axis on the interval, and is negative if it lies below the horizontal axis on the interval.
