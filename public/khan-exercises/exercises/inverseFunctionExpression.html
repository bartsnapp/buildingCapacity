f(x) = Ax+B

Find the formula for f^{-1}(x)

(x-B)/A

The definition of inverse functions states that f(f^{-1}(x))=x

So A\cdot f^{-1}(x)+B = x

A\cdot f^{-1}(x)= x-B

Therefore f^{-1}(x)= \dfrac{x-B}{A}

Write this as (x-(B))/(A)

f(x) = \dfrac{Ax+B}{Cx+D}

Find the formula for f^{-1}(x)

(Dx - (B))/(A - (Cx))

The definition of inverse functions states that f(f^{-1}(x))=x

So \dfrac{A\cdot f^{-1}(x)+B}{C\cdot f^{-1}(x)+D} = x

A\cdot f^{-1}(x)+B= x \cdot (C \cdot f^{-1}(x) + D)

A\cdot f^{-1}(x)+B=Cx\cdot f^{-1}(x) +Dx

(A -Cx)\cdot f^{-1}(x)+B=Dx

(A -Cx)\cdot f^{-1}(x)=Dx -B

f^{-1}(x)=\dfrac{Dx -B}{A -Cx}

Write this as (Dx - (B))/(A - (Cx))
